# Woodler - Optimization Scripts
In this directory you will find multiple implementations 
of the optimization model for the Woodler use case.

## Directory guide
* `woodler_cplex.py` - The most basic implementation
    of the optimization model that uses CPLEX.
* `woodler_grb.py` - The most basic implementation
    of the optimization model that uses Gurobi.
* `woodler_pulp.py` - The most basic implementation
    of the optimization model that uses PuLP.
